TipoNomeDescrição
Headerx-api-keyChave de API.

Ambiente: DEV
curl --request GET \
--url 'https://api.dev.exemplo.com/parametrizador/v1/products?segment=all' \
--header 'Authorization: Bearer {token}' \
--header 'Content-Type: application/json' \
--header 'x-api-key: {sua-chave}' \
--header 'x-correlation-id: {id-de-correlacao}'
