Level 3: Blastoise – Criando e resgatando ofertas

Chegou a hora de mostrar seu poder! Com tudo pronto, envie requisições e crie ofertas poderosas.

Parâmetros e CURLs por ambiente

  • Parâmetros
  • Request - dev
  • Request - hom
  • STS Token - dev
  • STS Token - hom
  • Headers
  • Body

**Tipo** | **Nome**           | **Descrição**
-------- | ------------------ | ------------------------------------------------------
Header   | `x-api-key`        | Chave de API necessária para autenticação.
Header   | `x-correlation-id` | ID de correlação para rastreamento de requisições.
Header   | `Authorization`    | Token de autenticação utilizado para validar e autorizar a requisição.
              

**Nome**                 | **Descrição**
------------------------ | -----------------------------------------------------
`storeNumber`           | Número da loja onde a oferta está sendo consultada.
`personIdentifier`      | Identificador único do cliente (CPF ou ID).
`products`              | Lista de produtos a serem consultados.
`familyProductCode`     | Código da família do produto.
`operationalProductCode`| Código operacional do produto.
`dn`                    | Número de identificação do produto.
`segmentCode`           | Código do segmento do produto.
              

curl --request POST \
--url https://api.dev.exemplo.com/ofertas/v1/consultar \
--header 'Authorization: Bearer {token}' \
--header 'Content-Type: application/json' \
--header 'x-api-key: {sua-chave}' \
--header 'x-correlation-id: {id-de-correlacao}' \
--data '{
"storeNumber": "000000000001",
"personIdentifier": "ID-do-cliente-aqui",
"products": [
  {
    "familyProductCode": 100,
    "operationalProductCode": 20001
  }
]
}'

curl --request POST \
--url https://sts.dev.exemplo.com/oauth2/token \
--header 'x-api-key: {sua-chave}' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--data-urlencode 'grant_type=client_credentials' \
--data-urlencode 'client_id={client-id}' \
--data-urlencode 'client_secret={client-secret}'
